Marker Markdown Editor for Windows

What is Marker?

Marker is a Markdown Editor for Windows. It's open source and currently under active development.

Download

You can download the latest executables in either zip or tar formats.

If you're interested, you can also download the source code with Git by running: 

$ git clone git://github.com/chrisledet/Marker

Support

If you happen to find a bug, please file it on our Issues page.
